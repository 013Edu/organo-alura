# Projeto-Organo-Alura

Um projeto feito através de aulas da Alura, onde basicamente criamos uma aplicaçao para organizarmos nossos times de professores da empresa de uma forma bem dinamica.

## Tecnologias usadas:

- HTML
- CSS
- Typescript
- ReactJs
